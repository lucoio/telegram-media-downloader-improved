# telegram-media-downloader-improved
Original script by: Nestor Qin  
First major update by: Finomosec  
Additional modifications by: Luishenriq1

I created this fork to implement additional improvements, 
fixes, and quality-of-life changes while preserving all 
original functionality.

What Has Been Changed
- Added support for topic-level media scraping  
- Improved download handling  
- Enhanced UI elements  
- Code cleanup and optimization  
- Fixed some Telegram Web compatibility issues

About:
This fork includes my own enhancements on top of the latest 
available version of the script.  
My goal is to improve usability, add missing features, and 
fix some issues found during real-world usage.

How to Install the Script
1. Install Tampermonkey (Chrome, Edge, Firefox).
2. Click the ".user.js" file in this repository.
3. Tampermonkey will automatically prompt installation.
